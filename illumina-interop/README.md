## How to use these notebooks?

* __Illumina InterOp report notebook:__ Use the template notebooks present under the `templates` directory to generate report of individual Illumina sequencing run. First generate a rendered notebook using the Jinja template using these codes. Then execute the notebook file within the container image which was built using the 'Dockerfile'.

```python
import os
from jinja2 import Environment, FileSystemLoader, select_autoescape
template_file = 'templates/interop_report_template_v1.ipynb'
rendered_template = 'report.ipynb'
data = dict(
    DATE_TAG='date_tag',
    RUN_ID='run_id',
    RUN_DIR='run_dir_path',
    METRICS_DIR='output_metrics_dir_path',
    OVERVIEW_CSV_OUTPUT='overview_csv_output_path',
    TILE_PARQUET_OUTOUT='tile_parquet_output_path',
    NUM_CPU='num_cpu',
    RAM_GB='ram_gb')
template_env = \
    Environment(
        loader=FileSystemLoader(
            searchpath=os.path.dirname(template_file)),
            autoescape=select_autoescape(autoescape_list))
template = \
    template_env.\
        get_template(
            os.path.basename(template_file))
template.\
    stream(**data).\
        dump(output_file)
```

* __Training notebook__: An example notebook for model training can be found in the 'example_notebooks' directory. It uses the 'tile_parquet_output_path' file generated by previous step (for individual runs) and a label of sequencing run status to train multiple models and check their qualities.
